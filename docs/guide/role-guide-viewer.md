# Viewer 가이드

Viewer는 갤러리에서 페이지를 구독하고, 실제 업무에 사용하는 소비자입니다. 페이지를 만들 필요 없이 원하는 어드민을 바로 사용할 수 있습니다.

## 해야 할 것

### 1. 갤러리 탐색 & 구독

갤러리에서 필요한 페이지를 검색하고 구독하면 사이드바 메뉴에 추가됩니다.

→ [페이지 갤러리](/guide/page-gallery)

### 2. 개인 메뉴 구성

구독한 페이지를 폴더별로 정리하여 나만의 사이드바를 만듭니다. 다른 사용자의 메뉴와 완전히 독립적입니다.

→ [메뉴 관리](/guide/menu-management)

### 3. 페이지로 업무 수행

구독한 페이지에서 데이터를 조회하고 수정합니다. 주문 검색, 상태 변경, 회원 정보 조회 등 실제 업무를 수행하세요.

::: info 페이지 사용 ≠ 페이지 관리
구독한 페이지 내에서 데이터 조회·수정(API 호출)은 정상 수행할 수 있습니다. 예를 들어 주문 목록에서 주문을 검색하고 상태를 변경하는 것은 "페이지 사용"이며, Viewer 권한으로 가능합니다.
:::

## 할 수 있는 것

### 갤러리 검색 & 미리보기 & 구독

공개된 모든 페이지를 검색하고, 미리보기로 확인한 뒤 구독할 수 있습니다.

### 사이드바 메뉴 자유 구성

폴더 생성, 순서 변경, 이름 변경 등 사이드바를 자유롭게 구성할 수 있습니다.

### 페이지 내 데이터 조회·수정

구독한 페이지 안에서 API를 통해 데이터를 조회하고 수정할 수 있습니다. 페이지가 제공하는 모든 기능을 사용할 수 있습니다.

## 장점

### IT 티켓 없이 원하는 어드민을 바로 사용

갤러리에서 구독 버튼 한 번이면 바로 사용할 수 있습니다. 별도의 권한 요청이나 설치가 필요 없습니다.

### 나만의 어드민 메뉴

다른 사용자와 독립적으로 메뉴를 구성합니다. 내 업무에 필요한 페이지만 골라서 효율적인 작업 환경을 만드세요.

### 항상 최신 버전

Editor가 페이지를 업데이트하면 구독자에게 자동 반영됩니다. 버전을 확인하거나 수동으로 업데이트할 필요가 없습니다.

### 보안 걱정 없음

모든 페이지는 iframe sandbox, CSP, OAS3 allowlist로 자동 격리됩니다. 별도의 보안 설정 없이 안전하게 사용할 수 있습니다.
