# 페이지 갤러리

다른 사용자가 만든 페이지를 탐색하고, 구독하여 자기 메뉴에 추가할 수 있습니다.

## 페이지 공개 상태

| 상태 | 설명 |
|------|------|
| `draft` | 작성 중. 본인만 접근 가능 |
| `private` | 완성됨. 본인만 사용 |
| `published` | 갤러리에 공개. 다른 사용자가 구독 가능 |

페이지를 저장할 때 또는 이후 설정에서 공개 상태를 변경할 수 있습니다.

## 갤러리 탐색

사이드바의 **갤러리** 메뉴에서 공개된 페이지를 탐색합니다.

갤러리는 카드 그리드 형태로 공개된 페이지를 보여줍니다:

- **검색**: 키워드로 페이지 검색
- **카드 정보**: 페이지 제목, 작성자, 사용 API 스펙
- **액션 버튼**: `[미리보기]` `[구독]`

**미리보기**를 클릭하면 실제 페이지 화면을 확인할 수 있습니다.

## 구독 (Subscribe)

원본 페이지를 **참조**하여 자기 메뉴에 추가합니다.

- 원작자가 페이지를 업데이트하면 **자동으로 반영**됩니다
- 직접 수정할 수 없습니다 (원본을 그대로 사용)
- 필요 없어지면 구독 해제 가능

## 구독 과정

1. 갤러리에서 원하는 페이지의 **구독** 버튼 클릭
2. 메뉴 위치 선택:

> **메뉴에 추가 다이얼로그**
>
> - **위치**: 메뉴 트리에서 추가할 폴더 선택
>
> `[추가]`

3. 사이드바에 해당 페이지 메뉴가 추가됩니다

## 내가 공유한 페이지 관리

**설정 > 내 페이지**에서 본인이 만든 페이지의 공개 상태를 관리할 수 있습니다.

- 구독자 수 확인
- 공개/비공개 전환
- 페이지 삭제 (구독자에게 알림)
