# 메뉴 관리

AdminCraft에서 각 사용자는 독립된 사이드바 메뉴 트리를 가집니다. 직접 만든 페이지와 구독한 페이지를 자유롭게 조합하여 자기만의 어드민을 구성합니다.

## 메뉴 구조

메뉴는 트리 구조로, 폴더와 페이지를 중첩할 수 있습니다.

```
📁 주문 관리
  ├── 📄 주문 목록        ← 직접 만든 페이지
  ├── 📄 주문 상세        ← 직접 만든 페이지
  └── 📄 주문 수정        ← 직접 만든 페이지
📁 회원 관리
  └── 📄 회원 목록        ← 갤러리에서 구독한 페이지
📁 대시보드
  └── 📄 매출 현황        ← 갤러리에서 구독한 페이지
```

## 메뉴 편집

### 폴더 추가

사이드바 하단의 **메뉴 편집** 버튼을 클릭하여 편집 모드로 진입합니다.

- **폴더 추가**: 새로운 그룹을 만듭니다
- **아이콘 변경**: 폴더나 페이지의 아이콘을 변경합니다

### 순서 변경

드래그 앤 드롭으로 메뉴 항목의 순서와 위치를 변경합니다.

- 항목을 다른 폴더로 이동
- 폴더 간 순서 변경
- 폴더 내 항목 순서 변경

### 메뉴에서 제거

메뉴에서 항목을 제거합니다.

- **직접 만든 페이지**: 메뉴에서만 제거 (페이지 자체는 유지)
- **구독한 페이지**: 구독 해제 + 메뉴에서 제거

## 페이지 추가 경로

메뉴에 페이지를 추가하는 방법은 두 가지입니다:

| 방법 | 설명 |
|------|------|
| **새 페이지 만들기** | AI 또는 코드로 페이지를 만들고 저장 시 메뉴에 등록 |
| **갤러리에서 구독** | 다른 사용자의 페이지를 구독하여 메뉴에 추가 |

## 일상적인 사용

메뉴 편집이 끝나면 일반 어드민과 동일하게 사용합니다.

- 사이드바에서 메뉴 클릭 → 해당 페이지 로드
- 페이지 간 이동은 URL 기반
- 사용자 페이지는 iframe 안에서 실행되지만, 사용자는 이를 인지하지 못합니다
